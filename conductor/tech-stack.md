# Tech Stack

## Core Technologies
- **Python:** The primary programming language used for the Home Assistant integration.
- **Home Assistant Core:** The framework and environment for which this component is built.

## Libraries & Dependencies
- **NumPy:** Used for efficient numerical operations, signal processing, and shape-correlation matching of power profiles.
- **Pytest:** The testing framework used for unit and integration tests.

## Development & Deployment
- **HACS (Home Assistant Community Store):** The primary distribution method for the integration.
- **GitHub Actions:** Used for CI/CD, including Hassfest validation and HACS compatibility checks.
